# Project Overview

This project demonstrates the development of an Industrial Human Machine Interface (HMI) system using a Nextion Touch Display and Arduino Mega 2560.

The HMI acts as a user-friendly control panel, allowing operators to interact with embedded hardware through a graphical touchscreen interface. Communication between the display and microcontroller is achieved through UART serial communication.

The system receives touch event commands from the HMI, processes them in real time, and controls connected devices accordingly.

This architecture is commonly used in industrial automation, smart control panels, manufacturing equipment, and IoT control systems.
